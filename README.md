# 🛢️ Crude Oil Production Analysis
## 📖 Project Overview

This project analyzes global crude oil production data using Python, Pandas, and Matplotlib. The dataset was cleaned, explored, and visualized to identify production trends across different countries and years.

The project demonstrates the complete data analysis workflow, including data cleaning, exploratory data analysis (EDA), visualization, and drawing meaningful insights from real-world data.
## 🎯 Project Objectives

- Understand the structure of the crude oil production dataset.
- Clean the dataset by handling missing values and removing unnecessary columns.
- Explore production trends using Exploratory Data Analysis (EDA).
- Visualize the data using basic charts in Matplotlib.
- Draw meaningful insights from the analysis.

## 📊 Exploratory Data Analysis

The following questions were explored during the analysis:

1. How many countries are included in the dataset?
2. Which are the top 10 crude oil producing countries?
3. How has global crude oil production changed over time?
4. How is crude oil production distributed across the dataset?
5. Which years recorded the highest crude oil production?
## 💡 Key Findings

- The dataset contains crude oil production records from many countries over several decades.
- A few countries contribute a major share of global crude oil production.
- Overall crude oil production has shown an increasing trend over time.
- Most production values are relatively low, while
 only a small number of observations represent very high production levels.
- The highest production values are concentrated in the later years of the dataset.
